How to Use this Template

To edit this document, Right Click here then choose Edit Preview Document, then an HTML editor.

Once you've finished, make sure you save, then in Style Master, click the Update Preview button below to see your changes in this preview.

  1. Replace the text yoursite.com with the name of your site - it's inside the <title> element in the <head> of the document, as well as inside an <h1> element directly inside <div id="banner">.
  2. This template has a meta navbar, which is inside a <div id="nav-meta">. You'll need to link this up to your contact, search and sitemap pages if you have them.
  3. Put the names of the major sections of your site into <div id="nav-main">. You'll then need to link all these up appropriately.
  4. Put the names of the subsections of this section into <div id="nav-section">. These will then need to be linked up appropriately as well.
  5. Put any additional links into <div id="sidebar">.
  6. Put your own links into the footer - this is an element called <div id="info-site">.
  7. Put your own text into all the paragraphs.
  8. Put any additional text below the links in <div id="sidebar">.
